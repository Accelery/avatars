Privacy Policy

Last updated: May 2026

The short version

No accounts, no cookies, no ads, no personal data. The API logs basic anonymous usage stats. Read on for details, or add ?notrack=true to any request to skip logging entirely.

What we receive

Every HTTP request carries an IP address and a URL. We have no control over that.

  • IP addresses pass through Cloudflare's network and are not stored by us.
  • The ID (the part after the / in the URL) is used to generate the avatar and shows up in our usage stats.

Usage stats

Each request logs:

  • The ID from the URL (e.g. username in api.genavatar.me/username). Requests with no ID are logged as random.
  • The referring page, if the browser sent one. That is the URL of the page where the avatar is embedded. Left blank if not provided.
  • A timestamp.

No IP address or personal information is stored. Logs are kept for up to 3 months and then deleted.

Opting out

Add ?notrack=true to skip logging:

https://api.genavatar.me/username?notrack=true

The avatar is still returned, nothing is logged.

Cookies and tracking

No cookies. No ad or analytics scripts. No third-party tracking on genavatar.me or api.genavatar.me.

Hosting

Both the API and the landing page are hosted on Cloudflare (Workers and Pages). Cloudflare has its own data practices: Cloudflare's privacy policy. We have no access to that data.

This page loads fonts from Google Fonts, so your browser contacts Google's servers for that.
