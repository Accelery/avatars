Privacy Policy

Last updated: May 2026

The short version

genavatar.me does not collect, store, or share any personal data. There are no accounts, no cookies, no advertising, and no tracking of any kind. The API does record lightweight, anonymous usage counters — see below for exactly what that means and how to opt out.

What we receive

When you request an avatar, the API receives a standard HTTP request that includes your IP address and the ID string in the URL path. This is a normal part of how the internet works.

  • IP addresses are not logged or stored beyond the transient request lifecycle managed by Google Cloud infrastructure.
  • ID strings (the path you pass to the API) are used to generate the avatar image and are also recorded in the usage log described below.

Usage analytics

Each API request updates an anonymous usage record keyed on the ID string. The following fields are stored:

  • The ID string from the URL path (e.g. username in api.genavatar.me/username). Random requests (no path) are recorded under the key random.
  • A request counter — how many times that ID has been requested.
  • A last-access timestamp.
  • The Referer header, if present — the URL of the page that embedded the avatar.

No IP address, user agent, or any other identifying information is stored in these records. The purpose is solely to understand which IDs are active and how often the service is used in aggregate.

Opting out

You can skip the usage record entirely by appending ?notrack=true to any API request:

https://api.genavatar.me/username?notrack=true

When this parameter is present the avatar is generated and returned as normal, but nothing is written to the usage log.

Cookies & tracking

We set no cookies. We run no analytics scripts. We use no third-party tracking pixels or SDKs on either genavatar.me or api.genavatar.me.

Third-party infrastructure

The service runs on Google Firebase (Cloud Functions and Hosting). Google may collect infrastructure-level telemetry as described in Google's Privacy Policy. We have no access to this data.

This landing page loads fonts from Google Fonts, which involves a request to Google's servers.

Data retention

Usage records (ID string, counter, timestamp, referer) are stored indefinitely but contain no personal data. If you would like a specific ID's record removed, open an issue on GitHub and we will delete it.
